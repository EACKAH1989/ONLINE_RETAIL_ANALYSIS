# Analysis of Online Retail Transactions

This project analyzes the **Online Retail II** dataset to explore customer behavior, revenue trends, and transaction patterns across countries using R.

## Dataset

The dataset contains the following columns:

- Invoice
- StockCode
- Description
- Quantity
- InvoiceDate
- Price
- Customer ID
- Country

## Key Analysis

- Total number of unique customers  
- Total number of countries with transactions  
- Total revenue generated  
- Total number of orders  
- Revenue per country  

## Tools Used

- R  
- R packages: `dplyr`, `ggplot2`, `tidyverse`, `readxl`, `janitor`  
- R Markdown for reproducible analysis
